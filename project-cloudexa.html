A banking-grade data visualization platform for securely managing a network of ATMs

Role Designer & Developer
Year 2025
Client ESQ Data Solutions
Figma Vue.js Vite Pinia DevExtreme pgAdmin Power BI

Overview

Cloudexa is a custom banking-grade data visualization platform built for ESQ Data Solutions — a Kinective company — to help enterprise clients securely monitor and manage their ATM networks. The platform surfaces 8 key operational metrics in a clean, accessible interface, replacing a fragile Power BI setup that was consistently failing high-profile clients.

Problem

ESQ spent millions building out a network of ATMs, but the weakness fell in the client's ability to use it. Clients continuously ran into the same issues — and it was almost never a technical failure. It was a monitoring and maintenance failure. High-profile clients were making simple mistakes.

An error on the client side is an error on the provider.

Process

The first redesign — Cloudexa version 1 — was already an improvement over Power BI. It introduced a Configuration and Status Manager with a structured sidebar, instance management, and a cleaner layout. But it still fell short:

  • Dead Space. Too much unused space created more cognitive load and forced users to click through more menus to find information.
  • Outdated Colors. A system is only as good as it feels when used — the visual language felt dated.
  • No Breadcrumbs. Users struggled to identify where they were within the application hierarchy.

Solution

Cloudexa version 2 is a purpose-built dashboard that puts the most critical ATM health data front and center. The Instance Status Dashboard presents Memory Usage by Service, Hotfixes Over Time, TLS Protocols Balance, Ciphers Balance, and Service Status by Server — all in a single scannable view without tooltips or hidden menus.

The result is a platform tailored to each client's environment — deployable, maintainable, and built to prevent the simple mistakes that were costing ESQ and their clients the most.
